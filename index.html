HTML Word Template

Write in HTML, view it like a word-processed document.

Why it's cool:

  • Save default styles via stylesheets
    • Especially helpful for accessibility control: e.g., if your eyes are sensitive to bright-white pages
    • Also allows you to change the interface background (e.g., "theater mode"), which is not an option on all word processors
    • Save different stylesheets for different publication styles (e.g., APA, MLA) instead of looking up the header hierarchy every time
  • Portable: Write the text in the editor of your choice without worrying if it will hold up on a different system
    • Avoid the bulkiness of word processors
    • Set up your environment how you want it
  • Guidelines to tell you where to break up the text into a new page
  • When writing in HTML, you can comment out any parts you're hesitant about, hiding them from the document preview without having to delete them altogether
  • Speaking of comments, you can add revision-style comments with the <comment></comment> wrapper (for the text you're commenting on) and the <ctext></ctext> wrapper for the comment itself
  • Version control systems actually like HTML files: keep track of the changes you've made on manuscripts, etc.

How to use it

If you're familiar with HTML and CSS:

First, obtain the relevant files

Option 1: Clone the repository

git clone https://github.com/cranndarach/html-word-template.git

Option 2: Download just what you need

  • Hop over to the repository
  • Download either sample.html (which has sample text in case you like visual examples) or template.html (which has the important tags and comments, but is otherwise empty)
  • Download a stylesheet (e.g., css/stylesheet.css) and save it in a subdirectory called css/ in the same directory where you saved the .html file (that is, the .html files link to css/stylesheet.css)

Then, you can write in the .html file (I'd save it as a new file first) in an editor of your choice, and when you're ready to preview it, open it in a browser.
